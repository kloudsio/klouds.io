# Klouds.io

Written with **io.js**. Database is **MongoDB**.



```bash

# install global dependencies
npm install -g duo sane

# installs klouds.io dependencies
make install

# compiles the front-end
make build

# runs server and client. watches for changes using sane
make develop

```
